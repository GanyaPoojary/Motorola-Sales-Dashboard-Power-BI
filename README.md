 📊 Motorola Sales Dashboard (Power BI)

📌 Project Background

This dashboard was created as part of a **30-Day Power BI Course** by *SkillCourse (Satish Dhawale)*.
The project demonstrates my ability to apply data analysis and visualization concepts learned during the course.

🎯 Objective

To analyze mobile sales data and derive meaningful business insights using interactive dashboards.

📈 Key Insights

* 📌 Total Sales reached **769M**
* 📌 Highest sales observed in **March**
* 📌 **UPI** is the most preferred payment method
* 📌 **iPhone SE** is the top-selling model
* 📌 Sales are higher on **weekends**

📊 Dashboard Features

* 🌍 Sales by City (Map Visualization)
* 📅 Monthly Sales & Quantity Trends
* 📱 Sales by Mobile Model
* 💳 Payment Method Distribution
* ⭐ Customer Ratings Analysis
* 📆 Day-wise Sales Performance
